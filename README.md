# 🐱 DocCat

**DocCat** is a lightweight **static site generator (SSG)** built with **React and TypeScript**.

It lets you write pages in **TSX** and **custom Markdown (`.dmd`)**, and compiles everything into **pure static HTML and CSS**.

No runtime. No hydration. Just fast websites.

---

## ✨ Features

- ⚡ **Pure Static Output** – Zero client-side JavaScript  
- ⚛️ **TSX as Templates** – Use React at build time  
- 📝 **`.dmd` Support** – Built-in format for blogs & documentation  
- 🧩 **Layouts** – Reusable page structure  
- 🎨 **Automatic Tailwind Support** – Detected and built automatically  
- 🖥 **CLI First** – Simple commands, no setup hassle  
- 📦 **Deploy Anywhere** – Works on any static hosting
